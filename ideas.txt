Colour Palette Picker:  Input a start colour (colour hex code), a medium-ish colour, and an end colour. The program should dynamically generate a palette of colours between those two points.
Number Pattern Generator: User enters a number (e.g. "5"). The program generates a sequence of 3 numbers where each successive number is the previous number plus twice its current value (repeated until you reach the initial number). Display the results.
Password Maker: Prompt user to provide a password of length at least 8 characters, including 1 uppercase letter, one digit and a special symbol. The program creates a random password, and then it reveals the final pattern to the user to find out how many times each character appears. This can be made much more difficult if input constraints are too tight.
Simple Number Sum:  Take two numbers as inputs (e.g. 10, 3). The program outputs the sum of those numbers. Repeat this several times and give a single number that the user must repeat at least once to find out the final result.
Palindrome Generator: Generate palindromes, creating them if given one number for an iterative function - repeat it and then ask a new number as input. 128 is an example for this task with multiple iterations.  Focus on just making those that meet the constraint easily.
Basic Math Challenge (Addition/Subtraction): The user inputs two numbers, or a calculation such as 3 + 7, the program prints out the result of it. Repeat this over several challenges like 'add two things', 'subtract anything'.
String to Integer Converter: Accept a string input and output an integer representation from the string.
Number guessing game: The program generates a number from a range (e.g. 1-100). The user attempts to guess what the number is, and the program will say higher or lower. You could also add a hard more where it only says yes or no to keep the game fun for longer!
To-do list: The user can create tasks (probably on a web interface) and the program will store the values so the user can mark them off or view them later. For a challenge, you could add priorities or encryption. Maybe even user accounts!
BMI calculator: The user inputs their weight and height, and the program will calculate the user's BMI from these values. You will have to figure out what formula you need to use, and whether you need to do any unit conversions.
Unit converter: The user enters a temperature or other unit and the program converts it into another. You will need to find the formulas for many different types of units and figure out how to easily convert. This idea could also work for currencies, where you would learn to ping an API to get up-to-date currency information.
Calculator: Takes two numbers and an operation to find the result.
Random Quote App: Build an app that displays a new quote whenever the user clicks a button. Start with a local list of quotes, then improve it by loading quotes from an online API. You can also add features like filtering by author, saving favorite quotes, or sharing quotes.
Age Calculator: Create a tool where users enter their date of birth and get their exact age in years, months, and days. You can make it more advanced by showing the total number of days lived, the weekday they were born on, and how many days remain until their next birthday.
Countdown Timer: Let the user input a number of seconds (or a future date/time) and display a live countdown. Add sound alerts, pause/resume buttons, and different timer themes. A good extra challenge is supporting multiple timers running at once.
Rock Paper Scissors: Make a game where the user plays against the computer, which makes random choices. Track wins, losses, and draws across rounds. You can extend it with animations, a “best of 5” mode, or a harder AI that learns from player history.
Grade Calculator: Ask the user to enter scores for assignments, quizzes, and exams, then calculate weighted totals and final grades. You can include grade boundaries (A, B, C, etc.), pass/fail status, and “what-if” analysis (e.g., “What do I need on the final exam to get an A?”).
Expense Tracker: Build a simple finance tracker where users add income and expense entries with category labels (food, travel, rent, etc.). Show total spending, remaining balance, and monthly summaries. You could add charts and export data to CSV for extra practice.
Prime Number Checker: Let users enter a number and check whether it is prime. Show the reason (for example, “not prime because divisible by 3”). A more advanced version could list all prime numbers in a range and compare faster vs slower checking methods.
Fibonacci Generator: Allow users to choose how many Fibonacci numbers to generate, then display the sequence clearly. You can also compare iterative and recursive methods to show performance differences. A visual graph of sequence growth would make a great extension.
Word Counter: Create a text utility that counts words, characters, sentences, and paragraphs. Make it update in real time as the user types. You can add useful writing metrics like estimated reading time or keyword frequency.
Anagram Checker: Ask for two words or phrases and check if they are anagrams by comparing normalized letters. Improve it by ignoring spaces, punctuation, and capitalization. A fun extra is generating possible anagrams from a dictionary list.
Dice Roller Simulator: Simulate rolling one or more dice and show individual values plus total score. This is a good project for practicing randomness and loops. You can add special dice (D20, D12, etc.), roll history, and simple probability summaries.
Contact Book: Create an app that stores contact details such as name, phone number, and email. Include options to add, edit, delete, and search contacts. Later, add validation (correct email format), categories, and persistent storage using files or a database.
Flashcard Quiz App: Build a learning app where users can create flashcards and test themselves. Include “show answer,” score tracking, and incorrect-answer review mode. You can make it more useful by adding spaced repetition and multiple topic decks.
Typing Speed Test: Show users a sample paragraph and measure typing speed (WPM) and accuracy. Include a timer and compare typed text against the original. You can improve it with difficulty levels, leaderboards, and error heatmaps.
File Organizer Script: Write a script that scans a folder and automatically moves files into categorized subfolders (images, documents, videos, etc.). Add safety checks to avoid duplicates and create logs of moved files. A strong extension is watching a folder continuously for new files.
Habit Tracker: Build an app where users add daily habits (reading, exercise, water intake) and mark them complete each day. Show streaks, completion percentages, and weekly progress. A useful extension is reminder notifications and a calendar view.
Weather Dashboard: Let users search for a city and see current weather plus a short forecast. Display temperature, humidity, wind, and weather conditions clearly. You can add unit switching (°C/°F), recent search history, and weather icons.
Notes App with Tags: Create a note-taking tool where users can add, edit, delete, and search notes. Support tags to organize notes by topic. You can extend it with markdown formatting, pinned notes, and local/cloud sync.
Pomodoro Focus App: Build a timer that cycles through focus sessions and short breaks (for example, 25/5 minutes). Track completed sessions and total focus time per day. Add sounds, task labels, and productivity charts.
URL Shortener (Local Version): Make a simple system that maps long links to short custom codes. Users input a URL and get a short code that redirects correctly. You can add click counters, expiry times, and duplicate-link detection.
Quiz Game Engine: Build a reusable quiz system that loads questions from a file or JSON object. Support multiple-choice questions, scoring, and final result summaries. Add time limits, random question order, and category selection.
Markdown Previewer: Create an editor where users type markdown on one side and see formatted output on the other. Update the preview in real time. You can include syntax highlighting, export to HTML, and dark/light themes.
Image Resizer Tool: Build a utility that allows users to upload an image and resize it by fixed dimensions or percentage. Keep aspect ratio options and preview the output before saving. Extra features can include compression and format conversion (PNG/JPG/WebP).
PDF Merger/Splitter: Let users combine multiple PDFs into one file or split one PDF into smaller files. Provide a clear interface to reorder pages before export. You can improve it with drag-and-drop and page thumbnail previews.
Recipe Finder App: Store a list of recipes and let users search by ingredient, cooking time, or category. Show step-by-step instructions and ingredient quantities. A strong extension is meal planning and grocery list generation.
Attendance Tracker: Build a small system for tracking attendance by name and date. Show attendance percentages automatically and highlight low attendance cases. Add CSV export and monthly reports as an advanced feature.
Currency Tracker: Create an app where users can track selected currencies against one base currency. Show conversion values and trend changes over time. You can add charts and alerts when rates cross a threshold.
Flash Sale Price Calculator: Let users enter original price, discount percentage, and tax rate to get final payable amount. Show each step of the calculation so users understand the result. Add support for multiple discounts and coupon stacking.
Simple Chat App (Local Network): Build a basic messaging app where two or more users can send text messages in real time. Start with one chat room, then add usernames and timestamps. Extensions include multiple rooms and message history storage.
Mind Map Creator: Let users create connected idea nodes visually (for study planning or brainstorming). Support adding/removing nodes and drawing links between them. A nice upgrade is saving and loading map structures in JSON format.
Invoice Generator: Build a tool where users enter client details, items, quantities, and prices to generate a professional invoice. Automatically calculate subtotal, tax, and total. Add PDF export and invoice number auto-increment.
Poll/Voting App: Create an app where users can create polls with multiple options and vote once. Display live vote counts and percentages. Add authentication, poll closing times, and duplicate-vote prevention.
System Resource Monitor: Build a dashboard that displays CPU, memory, and disk usage in near real time. Show values using progress bars or charts. You can add thresholds and alerts for high usage events.
Regex Tester Tool: Create a utility where users type a regular expression and test it against sample text. Highlight matching segments and explain errors if the regex is invalid. Add a library of common regex examples.
Text Encryption Playground: Build a simple tool to encode/decode text using methods like Caesar cipher or Base64 (for learning purposes). Let users pick a method and key where applicable. Add side-by-side input/output and copy buttons.
